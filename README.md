# WTD | What To Do - Life Calendar

## INSTALLATION

### Automatic installation
```bash
git clone https://github.com/Live-Proggress-Studio/WTD
cd WTD
make
```

P.s \
To stop server, please use `make stop` command

### Manual installation
```bash
git clone https://github.com/Live-Proggress-Studio/WTD
cd WTD
cd frontend
yarn && yarn dev
cd ../backend
make
```
